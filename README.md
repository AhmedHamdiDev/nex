# 🚀 NEX

**NEX** is a lightweight developer tool that runs C, Java, and Python files or projects with a single command.

It detects project structure automatically, supports configuration files, and measures execution time.

Built for speed, simplicity, and productivity.

---

# ✨ Features

- Run Python, C, and Java files
- Run entire projects with `run .`
- Automatic project detection
- `nex.config` support (custom entry file)
- Execution time measurement
- Simple command-line interface
- Modular architecture
